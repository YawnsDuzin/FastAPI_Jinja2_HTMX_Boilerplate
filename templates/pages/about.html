소개

이 프로젝트는 FastAPI, Jinja2, HTMX를 결합한 모던 풀스택 웹 애플리케이션 보일러플레이트입니다. JavaScript 프레임워크 없이도 동적인 SPA-like 경험을 구현할 수 있습니다.

프로젝트 특징

제로 JavaScript 빌드

HTMX와 Alpine.js를 사용하여 별도의 빌드 프로세스 없이 동적 UI를 구현합니다.

서버사이드 렌더링

SEO 친화적인 서버사이드 렌더링으로 검색 엔진 최적화가 용이합니다.

빠른 UI 업데이트

Partial HTML 응답으로 필요한 부분만 업데이트하여 빠른 사용자 경험을 제공합니다.

모듈화 구조

확장과 유지보수가 쉬운 깔끔하게 분리된 프로젝트 구조를 제공합니다.

기술 스택

구분 기술 용도
Backend FastAPI 비동기 웹 프레임워크
Template Jinja2 서버사이드 렌더링
Frontend HTMX AJAX/동적 UI
CSS TailwindCSS 스타일링
Database SQLAlchemy ORM
Auth python-jose JWT 인증
